# Experiments

Use one folder per experiment ID.

## Recommended Lifecycle
1. Copy `exp_template/` into a new directory named `exp_###_<slug>`.
2. Fill in metadata.
3. Record commands and outputs.
4. Link outputs in `results/` and interpretation in `analysis/`.

## Core Files
- `experiment_registry.md`: quick status tracker for all experiments
- `exp_template/`: starter folder and metadata template
