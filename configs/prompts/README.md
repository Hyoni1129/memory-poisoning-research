# Prompt Artifacts

Store public, reproducibility-relevant prompts here when they are shared across experiments or pipeline stages.

Use experiment-local prompt folders for prompts that belong to one experiment:

```text
experiments/<exp_id>/prompts/
```

For each prompt artifact, include:
- purpose
- model or tool used
- date first used
- linked experiment or dataset ID
- expected input and output format
- verification status or known caveats
